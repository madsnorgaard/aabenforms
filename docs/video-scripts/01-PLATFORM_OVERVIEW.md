# Video Script 01: Platform Overview (3 minutes)

## Video Metadata
- **Duration**: 3 minutes
- **Target Audience**: Municipal decision-makers, IT managers
- **Goal**: Introduce ÅbenForms and establish value proposition
- **Tone**: Professional, confident, solution-focused

---

## Technical Setup

### Recording Environment
- Clean desktop background (solid color or municipal logo)
- Browser: Chrome/Firefox in full-screen mode (F11)
- Resolution: 1920x1080 minimum
- Screen recording at 30fps

### Pre-recording Checklist
- Clear browser cache and cookies
- Close unnecessary browser tabs
- Disable notifications (Do Not Disturb mode)
- Test microphone levels
- Prepare demo environment at: https://demo.aabenforms.dk

### Visual Assets Needed
- ÅbenForms logo (high-res)
- Cost comparison chart
- Feature comparison table
- Danish municipal logos (anonymized)

---

## SCRIPT

### [0:00-0:15] Opening (15 seconds)

#### NARRATION (Danish)
"Velkommen til ÅbenForms - Danmarks første open source digitale selvbetjeningsplatform, bygget specifikt til danske kommuner."

#### NARRATION (English)
"Welcome to ÅbenForms - Denmark's first open-source digital self-service platform, built specifically for Danish municipalities."

#### SCREEN ACTIONS
1. Start with black screen
2. Fade in ÅbenForms logo (centered)
3. Transition to homepage: https://demo.aabenforms.dk
4. Show clean, modern interface

#### VISUAL CALLOUTS
- Highlight logo and tagline
- Show "Open Source" badge prominently

#### RECORDING TIPS
- Use smooth fade transitions (1 second)
- Background music: subtle, professional (low volume)
- Speak clearly and with enthusiasm

---

### [0:15-0:45] The Problem Statement (30 seconds)

#### NARRATION (Danish)
"I dag betaler danske kommuner millioner af kroner årligt for digitale selvbetjeningsløsninger som KMD XFlow. Men hvad nu hvis der fandtes en moderne, open source alternative - uden vendor lock-in, uden skjulte omkostninger, og med fuld kontrol over jeres data?"

#### NARRATION (English)
"Today, Danish municipalities pay millions of kroner annually for digital self-service solutions like KMD XFlow. But what if there was a modern, open-source alternative - without vendor lock-in, without hidden costs, and with full control over your data?"

#### SCREEN ACTIONS
1. Show cost comparison slide:
   - XFlow: 500,000 kr/year (typical municipality)
   - ÅbenForms: Self-hosted (free) or 50,000 kr/year (managed)
2. Highlight savings: **90% cost reduction**

#### VISUAL CALLOUTS
- Animated bar chart showing cost comparison
- Highlight "90% savings" in bold
- Show checkmarks for benefits:
  -  No vendor lock-in
  -  Full data sovereignty
  -  Community-driven development

#### RECORDING TIPS
- Pause for 2 seconds on cost comparison chart
- Use arrow annotations to highlight key numbers
- Emphasize "millions of kroner" with slight vocal stress

---

### [0:45-1:30] Key Features Overview (45 seconds)

#### NARRATION (Danish)
"ÅbenForms giver jer alle de funktioner I kender fra XFlow - og meget mere. MitID-integration, betalingshåndtering via Nets, PDF-generering, SMS-beskeder, og kompleks workflow-orkestrering. Men vi går længere: GIS-integration til byggesager, GDPR-compliance by design, og en visuel workflow-builder der gør det let at oprette nye selvbetjeningsløsninger uden at skrive kode."

#### NARRATION (English)
"ÅbenForms provides all the features you know from XFlow - and more. MitID integration, payment processing via Nets, PDF generation, SMS notifications, and complex workflow orchestration. But we go further: GIS integration for building permits, GDPR compliance by design, and a visual workflow builder that makes it easy to create new self-service solutions without writing code."

#### SCREEN ACTIONS
1. Navigate to Features page
2. Scroll through feature list slowly
3. Show feature comparison table:

| Feature | XFlow | ÅbenForms |
|---------|-------|-----------|
| MitID Integration |  |  |
| Payment Processing |  |  |
| PDF Generation |  |  |
| SMS Notifications |  |  |
| GIS Zoning Validation | ✗ |  |
| Visual Workflow Builder | Limited |  |
| Open Source | ✗ |  |
| Self-Hosting | ✗ |  |

#### VISUAL CALLOUTS
- Highlight each feature with animated checkmark
- Circle "GIS Integration" as unique feature
- Box "Visual Workflow Builder" section
- Use green highlights for ÅbenForms advantages

#### RECORDING TIPS
- Slow down when listing technical features
- Use cursor to point to each feature as you mention it
- Pause briefly between feature categories

### [1:30-2:15] Real-World Use Cases (45 seconds)

#### NARRATION (Danish)
"Lad os se på konkrete eksempler. Med ÅbenForms kan borgerne ansøge om parkeringslicenser, booke vielsesceremoni, indsende byggeansøgninger, og meget mere - alt sammen med samme brugervenlige oplevelse. Hver workflow kan tilpasses jeres kommunes unikke behov, og når I er klar, kan I dele jeres løsninger med andre kommuner i fællesskabet."

#### NARRATION (English)
"Let's look at concrete examples. With ÅbenForms, citizens can apply for parking permits, book marriage ceremonies, submit building permit applications, and much more - all with the same user-friendly experience. Each workflow can be customized to your municipality's unique needs, and when you're ready, you can share your solutions with other municipalities in the community."

#### SCREEN ACTIONS
1. Show workflow gallery page
2. Hover over each workflow card:
   - Parking Permit Application
   - Marriage Ceremony Booking
   - Building Permit Application
   - Food Establishment License
   - Dog Registration
3. Click into "Parking Permit" workflow (preview mode)
4. Show clean citizen-facing form

#### VISUAL CALLOUTS
- Label each workflow card with category
- Show "15 ready-to-use workflows" counter
- Highlight "Community Templates" section
- Show workflow complexity indicator (Simple/Medium/Complex)

#### RECORDING TIPS
- Smooth mouse movements when hovering
- Pause 2 seconds on parking permit form
- Use zoom effect to focus on form quality

---

### [2:15-2:45] Open Source Advantages (30 seconds)

#### NARRATION (Danish)
"Som open source projekt betyder ÅbenForms transparens, sikkerhed gennem community-review, og frihed til at tilpasse platformen præcis som I vil. Der er ingen sorte kasser, ingen leverandør-afhængighed. Jeres kommune ejer sin digitale infrastruktur."

#### NARRATION (English)
"As an open-source project, ÅbenForms means transparency, security through community review, and freedom to customize the platform exactly as you wish. There are no black boxes, no vendor dependency. Your municipality owns its digital infrastructure."

#### SCREEN ACTIONS
1. Show GitHub repository: https://github.com/aabenlforms/platform
2. Highlight key metrics:
   - Stars: 1,247
   - Forks: 89
   - Contributors: 23
   - License: AGPL-3.0
3. Show recent commit activity
4. Show documentation site

#### VISUAL CALLOUTS
- Animate GitHub stats counting up
- Highlight "23 contributors" (community strength)
- Box the AGPL-3.0 license badge
- Show "Active Development" badge with green pulse

#### RECORDING TIPS
- Screen capture GitHub page at 1920x1080
- Use browser zoom to make metrics readable
- Professional, trustworthy tone for security points

---

### [2:45-3:00] Call to Action (15 seconds)

#### NARRATION (Danish)
"Klar til at spare millioner og tage kontrol over jeres digitale selvbetjening? Besøg aabenforms.dk for at prøve vores demo, downloade platformen, eller kontakte os for en personlig demonstration. ÅbenForms - open source til danske kommuner."

#### NARRATION (English)
"Ready to save millions and take control of your digital self-service? Visit aabenforms.dk to try our demo, download the platform, or contact us for a personal demonstration. ÅbenForms - open source for Danish municipalities."

#### SCREEN ACTIONS
1. Transition to closing slide with:
   - ÅbenForms logo
   - Website: www.aabenforms.dk
   - Email: kontakt@aabenforms.dk
   - GitHub: github.com/aabenforms/platform
2. Show QR code linking to demo site
3. Fade to black

#### VISUAL CALLOUTS
- Large, readable contact information
- Animated QR code (subtle pulse effect)
- "Try Demo Now" button with arrow
- Social proof: "Join 12 municipalities already using ÅbenForms"

#### RECORDING TIPS
- Hold final slide for 5 full seconds
- Background music swells slightly at end
- Clear, confident closing tone
